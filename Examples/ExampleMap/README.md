# Red Cube Arena - Example Custom Map

This is a minimal example showing how to create a custom golf course.

## What This Example Shows

- Creating a Unity scene for golf
- Setting up Addressables for maps
- Configuring mod.json for map mods
- Building and exporting asset bundles

## Scene Contents

- **Red Cube** - A simple 2x2x2 red cube as visual indicator
- **Ground Plane** - Basic terrain
- **Lighting** - Default directional light

## Addressables Setup

The scene is added to Addressables with address: `CustomCourseRedCubeTest`

**Important:** All custom maps MUST have addresses starting with `CustomCourse`

## Building This Example

1. Open the scene in Unity
2. Window → Asset Management → Addressables → Groups
3. Verify scene is in a group with correct address
4. Window → Asset Management → Addressables → Build → New Build → Default Build Script
5. Find output in `Assets/AddressableAssetsData/Windows/`
6. Copy files to `[Game]/Mods/Maps/RedCubeArena/`

## Testing

1. Copy this folder to `[Game]/Mods/Maps/ExampleMap/`
2. Launch game with mod loader
3. The map should appear in custom courses
4. Load the map - you should see the red cube in-game!
